## Hypothesis

The local supervisor should expose a derived runner liveness status from durable heartbeat metadata. After a controller reconnects, a recent successful heartbeat should surface `runner_asleep` instead of being treated as missing, healthy, or empty. Missing, stale, disconnected, and failed states should be distinguishable without a live remote runner.
