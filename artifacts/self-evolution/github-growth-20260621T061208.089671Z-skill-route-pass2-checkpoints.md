bounded local checkpoints before activation. COMPASS-style state/profile lanes
need explicit handoff without profile or memory writes. FableCodex-style
workflow evidence needs replay checkpoints before closeout. Omnigent-style
management pressure supports a compact handoff surface instead of requiring
operators to infer next actions from scattered config or nested packets.
