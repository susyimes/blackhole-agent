## Hypothesis

but the broader `validation_readiness_summary` could still appear ready without
consulting that panel. A supervisor should not have to compare two surfaces to
know whether the selected local lane is replayable. The readiness summary should
consume the provider-runtime preflight result and report body-free recovery
hints directly when provider/runtime diagnostics are unresolved.
