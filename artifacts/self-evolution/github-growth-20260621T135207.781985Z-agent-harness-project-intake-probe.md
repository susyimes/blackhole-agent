## Hypothesis

probe before local behavior adoption. Recording install shape, entrypoints,
dependency boundaries, task-loop assumptions, and observable behaviors makes the
lane more useful than a claim map alone while still denying install, import,
external harness execution, provider launch, remote execution, and external
agent activation.
