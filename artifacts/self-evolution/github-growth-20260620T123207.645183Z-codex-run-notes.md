## Hypothesis

Skill-like public repositories should become bounded, operator-visible local
lanes before activation. The existing proposal rows were already bounded, but
reviewers had to reconstruct the candidate-level mapping from expanded rows.
Adding a compact candidate inventory to the deterministic lane map improves
preadmission inspection without adding install, execution, or activation
authority.
