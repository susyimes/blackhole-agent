and exposed it through the local harness output. The lane binds:

- `p1-skill-route-discovery-docs-and-probe` to the documentation lane.
- `p2-skill-route-discovery-test-fixtures` to the test lane.
- `p3-game-frontend-skill-profile-discovery` to the documentation lane for

activation.
