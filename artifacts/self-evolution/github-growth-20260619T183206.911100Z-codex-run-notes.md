## Evidence

- `https://github.com/omnigent-ai/omnigent/issues/851` reports that native sessions launched first through a Skill or slash-command can remain untitled and fall back to a generic provider label.
- The local reusable lesson is a controller-visible metadata invariant: skill-first native sessions must preserve enough command, skill, prompt, or launch context to seed a descriptive title before activation.

## Hypothesis

Adding a body-free local harness lane for native skill session titles improves runner workflow legibility end to end without launching external providers or exposing command, prompt, title, or session bodies.
