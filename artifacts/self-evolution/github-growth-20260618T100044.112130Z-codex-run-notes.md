## Hypothesis

Mock REPL/e2e approval fixtures can produce stale-green results if a governed tool-call step is supposed to request review but the local evaluator receives no approval verdict and treats it as `no_opinion`. Adding an explicit `approval_expected` contract makes this failure locally replayable without exposing session IDs, tool arguments, or tool names.
