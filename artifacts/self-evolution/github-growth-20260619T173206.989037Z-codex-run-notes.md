## Hypothesis

If `agent_workflow_route` can model streamed tool-call/tool-result boundaries as part of mid-flight runner state, future harness/control-plane changes will be easier to replay and recover. The local lane should reject complete tool results that are not strict JSON, reject URL evidence refs, preserve item-id-only citations, and avoid exporting raw stream bodies.
