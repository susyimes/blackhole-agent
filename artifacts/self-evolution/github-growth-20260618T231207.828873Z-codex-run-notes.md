## Hypothesis

Upgrade paths should expose a local, testable version decision before invoking package managers, installers, source updates,
or runner restart behavior. The Omnigent evidence documents an upgrade check and update command, plus alpha/pre-release
status, so the reusable local lesson is to make stable upgrade decisions explicit and keep development or preview upgrade
paths opt-in.
